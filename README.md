## Technologies
- **React**
- **Express**
- **Node.js**
- **CSS**
- **TailwindCSS**

Hosted on my homelab server using **Docker Compose** and **NGINX** reverse proxy.

## Running Developer Environment and Building

**Clone the git repo**
```
git clone https://github.com/shyamdhokia1/pomodoro-timer
```
**Run the dev environment**
```
npm run dev
```
**Build**
```
npm run build
```
**Docker Container**
```
docker compose up -d
```
